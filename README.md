## Project Summary

***PublicScripts*** is a public repository intended to host a collection of reusable scripts.

***What the project does***

- Provides a centralized location for publicly shareable scripts.
- Enables reuse of common automation, tooling, or utility logic.

***Why it exists***

- To avoid duplication of effort across projects.
